Python is one of the most useful programming language that is available today. Python is default language for data science and data engineering not to mention web development, scraping, scripting, and many other use cases.

This website will be a collection of resources from being assembled by the team at [Badass Courses](https://badass.dev) to bring clear, concise, in-depth, and comprehensive Python educational materials to working developers and teams that want to leverage the power and utility of this beautiful programming language.
